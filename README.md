### Usage
```sh
# For Rust
# example: cargo run --bin advent_2015_day_01
cargo run --bin advent_<year>_day_<day number>

# For Python
cd advent_<year>/src/python
# example: python3 advent_2015_day_01.py
python3 advent_<year>_day_<day number>.py
```
